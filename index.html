Cloud Computing Laboratory

Welcome to UNIJI Cloud Computing Laboratory!

The purpose of this practical session is to demonstrate how a simple HTML website can be published to the cloud.

Technology Used

  • HTML
  • Git
  • GitHub

Learning Objectives

  1. Create a GitHub repository.
  2. Upload project files to GitHub.
  3. Deploy the project using GitHub Pages.
  4. Access the deployed website through the Internet.

Expected Result

Students should understand the basic workflow of deploying a website to a cloud platform using GitHub.
