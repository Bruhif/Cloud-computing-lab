Cloud Computing Laboratory

Welcome!

This website has been deployed using Render. The purpose of this practical session is to demonstrate how a simple HTML website can be published to the cloud.

Technology Used

  • HTML
  • Git
  • GitHub
  • Render (Platform as a Service)

Learning Objectives

  1. Create a GitHub repository.
  2. Upload project files to GitHub.
  3. Deploy the project using Render.
  • Access the deployed website through the Internet.

    • Expected Result

    Students should understand the basic workflow of deploying a website to a cloud platform using GitHub and Render. Cloud Computing Laboratory 2026
